# Sound Match

A memory card game where you match sounds instead of images.

## Game Description

Flip cards to hear sounds and find matching pairs. Test your audio memory across increasingly challenging levels.

## Features

- Sound-based card matching gameplay
- Multiple difficulty levels
- Score tracking and high scores
- Hint system
- Clean, playful UI

## Tech Stack

- **Framework:** Flutter
- **Platform:** iOS (primary), Android (planned)
- **Design:** Pencil

## Getting Started

### Prerequisites

- Flutter SDK 3.38+
- Xcode (for iOS development)
- CocoaPods

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd musicmemo

# Get dependencies
flutter pub get

# Run on Chrome (for quick testing)
flutter run -d chrome

# Run on iOS Simulator
flutter run -d ios
```
